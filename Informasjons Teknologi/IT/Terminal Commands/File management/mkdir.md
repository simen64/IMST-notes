**Purpose:**  
Creates a new folder (directory).

Basically: _"Spawn a new room in the file system.".  


Commands


	mkdir foldername = create folder named foldername

	mkdir folder1 folder2 = Create multiple folders at once

	mkdir -p path/to/folder = Create nested directories (Creates all missing folders in the path)

	mkdir -m 755 foldername = Create folder and set permissions (Advanced - linux only)
