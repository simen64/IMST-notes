**Purpose:**  
`cd` moves you between folders (directories) in a terminal.  
Basically the _"take me somewhere else"_ command.

Think of it like teleporting between rooms in your file system.



**Commands**

	cd foldername  = Changing into foldername

	cd .. = Move up one directory (parent)

	cd ../.. = Move up two directorys

	cd /path/to/dir = change to an absolute path

	cd ~ = go to your home directory

	cd = go to your home directory 

	cd - = switch to previoucs directory
